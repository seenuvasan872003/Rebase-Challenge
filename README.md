# Rebase Challenge: MERN Stack (Stage 0)

Welcome to the Rebase Challenge. Your task is to fix a simple Todo application. The project is "broken" in a few places.

### Time Limit
This should take no more than **1 hour**.

### The Task
1.  **Clone** this repository (or copy the files).
2.  **Fix the Backend**:
    *   The database connection might be unstable.
    *   The `POST /api/todos` endpoint is currently failing to save the title correctly.
3.  **Fix the Frontend**:
    *   The "Add Task" button is sending data, but it's not appearing in the list without a refresh.
    *   Ensure the property names match between Frontend and Backend.
4.  **Host it**:
    *   Deploy the backend (e.g., Render, Fly.io, or Railway).
    *   Deploy the frontend (e.g., Vercel or Netlify).
5.  **Submit**:
    *   Paste your **GitHub Repository Link** and **Live Website Link** in the Rebase LMS dashboard.

### Setup
*   `cd backend && npm install && npm run dev`
*   `cd frontend && npm install && npm run dev`
*   Create a `.env` file in the backend with `MONGODB_URI`.
